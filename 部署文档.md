# 部署文档

## 1 相关依赖

* Redis 6.0.6
* Neo4j 3.5.28
* MySQL 8.0.22

## 2 项目启动

1. 启动数据库，导入 `web\src\main\resources\SQLScripts` 中的MySQL表结构

2. 启动 `KBQA/KBQA/KBQA_backend.py` Flask 服务

3. 配置 backend 的 `application.yml` 中关于数据库的配置

4. 启动后端 SpringBoot

5. 启动前端

  ```shell
  # install dependencies
  yarn install
  
  # serve with hot reload at localhost:8080
  yarn start
  
  # build for production with minification
  yarn build
  
  # run all tests
  yarn test
  ```
